My first blog

##开始
创建一个文件夹存放博客文件
首先要安装 node.js npm 两个服务
使用 npm 全局安装 hexo-cli 服务

###基本 hexo 命令使用
hexo s (start)启动本地博客，默认端口号: localhost:4000
hexo n (new)创建一个新的博客文章
hexo clean (清理)写完博客文章后，清理下数据
hexo g (刷新)对数据进行刷新
hexo s 重新启动 hexo 服务,可以看到 hexo 自动为你搭建了一个服务

###部署到 github 仓库
将 hexo 部署到 github 1.在自己的 github 上面搭建一个仓库，注意：仓库名要是自己的用户名.github.io 2.使用 npm 装一个 git 部署插件 命令：npm install -g -save hexo-deployer-git 3.修改配置 _config.yml 文件
在 deloyment 下面的 deploy 修改：
type: git
repo: (github 仓库地址)
branch: masten(指向空指针的意思) 4.将 hexo 部署导远端：hexo d (deplok)
注意：如果显示 not fround git 说明插件没安装好
window 电脑不需要再使用 github 用户密码登陆，直接推到远端
但是 linux 和 unix 相反

###修改主题，进行自定义
clone github 上面的主题到你所在的文件夹里面
修改主题文件还是 _config.yml 的 theme：landspace
